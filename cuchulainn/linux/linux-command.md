# Linux基礎コマンド、叩き台


cd：ディレクトリの移動

ls：ディレクトリの内容一覧表示

pwd：現在のディレクトリを表示

mkdir：新しいディレクトリを作成

rmdir：空のディレクトリを削除

rm：ファイルやディレクトリを削除

cp：ファイルやディレクトリをコピー

mv：ファイルやディレクトリを移動または名前変更

touch：新しいファイル作成やタイムスタンプ変更

cat：ファイルの内容を表示

less：ファイルをページ単位で表示

find：ファイルやディレクトリを検索

chmod：ファイルやディレクトリの権限変更

chown：ファイルやディレクトリの所有者変更

ps：現在のプロセス一覧表示

top：動作中のプロセスのリソース使用状況を表示

df：ディスク使用量を表示

free：メモリの使用状況を表示

ping：ネットワークの疎通確認

sudo：管理者権限でコマンド実行
