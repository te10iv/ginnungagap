# AWS CloudTrail：まずこれだけ（Lv1）

## このサービスの一言説明
- AWS CloudTrail は「**API操作ログの記録・監査**」を提供するAWSサービス

## ゴール（ここまでできたら合格）
- CloudTrail証跡を **作成できる**
- **API操作履歴を確認できる**
- 「監査ログにはCloudTrailが必要」と判断できる

## まず覚えること（最小セット）
- **証跡（Trail）**：ログ記録設定
- **イベント**：API呼び出し記録（誰が、いつ、何を）
- **S3保存**：ログの長期保存
- **管理イベント**：リソース操作（EC2起動等）
- **データイベント**：S3オブジェクト操作、Lambda実行等

## できるようになること
- □ 証跡を作成できる
- □ イベント履歴を確認できる
- □ 特定ユーザーの操作を追跡できる
- □ S3に保存されたログを確認できる

## まずやること（Hands-on）
- 証跡作成（S3バケット指定）
- イベント履歴確認（過去90日）
- EC2インスタンス起動・停止を記録確認
- CloudWatch Logs統合

## 関連するAWSサービス（名前だけ）
- **S3**：ログ保存先
- **CloudWatch Logs**：リアルタイム分析
- **EventBridge**：イベント駆動
- **Athena**：ログ分析（SQLクエリ）
- **GuardDuty**：セキュリティ脅威検知
- **IAM**：アクセス制御
