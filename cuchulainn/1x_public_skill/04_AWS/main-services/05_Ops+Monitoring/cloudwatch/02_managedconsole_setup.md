# CloudWatch マネージドコンソールでのセットアップ

## 作成するもの

CloudWatchアラームを作成し、EC2インスタンスのCPU使用率を監視します。CPU使用率が80%を超えた場合に通知を送信します。

## セットアップ手順

1. **CloudWatchコンソールを開く**
   - AWSマネージメントコンソールで「CloudWatch」を検索して選択

2. **アラームを作成**
   - 「アラーム」を選択
   - 「アラームを作成」をクリック

3. **メトリクスを選択**
   - 「メトリクスを選択」をクリック
   - **名前空間**: EC2を選択
   - **メトリクス名**: CPUUtilizationを選択
   - **インスタンス**: 監視するEC2インスタンスを選択
   - 「メトリクスを選択」をクリック

4. **メトリクスと条件を指定**
   - **アラーム名**: `high-cpu-alarm` を入力
   - **条件**: 静的を選択
   - **しきい値**: 80を入力
   - **比較**: より大きい（>）を選択

5. **通知を設定**
   - **通知**: SNSトピックを作成または既存のトピックを選択
   - **アラーム状態のトリガー**: アラーム状態を選択

6. **確認と作成**
   - 設定を確認
   - 「アラームを作成」をクリック

## 補足

- アラームは数分後に有効になります
- SNSトピックにメールアドレスをサブスクライブすると、通知を受け取れます
