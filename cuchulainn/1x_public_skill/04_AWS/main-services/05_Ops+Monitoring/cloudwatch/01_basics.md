# Amazon CloudWatch：まずこれだけ（Lv1）

## このサービスの一言説明
- Amazon CloudWatch は「**AWSリソースの監視・アラート**」を提供するAWSサービス

## ゴール（ここまでできたら合格）
- CloudWatchメトリクスを **確認できる**
- **アラームを作成できる**
- 「監視にはCloudWatchが必要」と判断できる

## まず覚えること（最小セット）
- **メトリクス**：数値データ（CPU使用率、ディスク使用量等）
- **アラーム**：閾値監視、SNS通知
- **ダッシュボード**：メトリクス可視化
- **標準メトリクス**：AWS自動収集（5分間隔）
- **カスタムメトリクス**：独自メトリクス送信

## できるようになること
- □ EC2のCPU使用率を確認できる
- □ アラームを作成できる（CPU > 80%でSNS通知）
- □ ダッシュボードを作成できる
- □ カスタムメトリクスを送信できる

## まずやること（Hands-on）
- EC2のメトリクス確認（CPU、ネットワーク）
- アラーム作成（CPUUtilization > 80%）
- SNSトピック作成、メール通知設定
- ダッシュボード作成

## 関連するAWSサービス（名前だけ）
- **SNS**：アラーム通知先
- **Lambda**：アラーム時の自動対応
- **EC2 / RDS / その他**：監視対象
- **CloudWatch Logs**：ログ監視
- **EventBridge**：イベント駆動
