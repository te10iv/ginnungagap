# CodeBuild マネージドコンソールでのセットアップ

## 作成するもの

CodeBuildプロジェクトを作成し、ソースコードをビルドします。GitHubやCodeCommitからソースを取得してビルドします。

## セットアップ手順

1. **CodeBuildコンソールを開く**
   - AWSマネージメントコンソールで「CodeBuild」を検索して選択

2. **プロジェクトを作成**
   - 「プロジェクトを作成」をクリック

3. **プロジェクト設定**
   - **プロジェクト名**: `my-build-project` を入力

4. **ソースを設定**
   - **ソースプロバイダー**: AWS CodeCommitを選択
   - **リポジトリ**: 既存のCodeCommitリポジトリを選択
   - **参照タイプ**: ブランチを選択
   - **ブランチ**: mainを選択

5. **環境を設定**
   - **環境イメージ**: マネージドイメージを選択
   - **オペレーティングシステム**: Amazon Linux 2を選択
   - **ランタイム**: Standardを選択
   - **イメージ**: aws/codebuild/amazonlinux2-x86_64-standard:5.0を選択
   - **サービスロール**: 新しいサービスロールを作成

6. **ビルド仕様を設定**
   - **ビルド仕様**: buildspec.ymlファイルを使用を選択

7. **確認と作成**
   - 設定を確認
   - 「プロジェクトを作成」をクリック

8. **ビルドを実行**
   - プロジェクトを選択
   - 「ビルドを開始」をクリック

## 補足

- buildspec.ymlファイルでビルド手順を定義します
- ビルドログはCloudWatch Logsに送信されます
