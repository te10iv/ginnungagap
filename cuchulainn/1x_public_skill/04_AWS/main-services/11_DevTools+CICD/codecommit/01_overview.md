# CodeCommit 概要

## 1. サービスの役割

AWS CodeCommitは、Gitベースのソースコード管理サービスです。プライベートなGitリポジトリをホストし、コードのバージョン管理、ブランチ管理、プルリクエストなどの機能を提供します。

## 2. 主なユースケース

- ソースコードのバージョン管理
- プライベートGitリポジトリのホスティング
- CI/CDパイプラインのソースコード管理
- コードレビューとコラボレーション
- ブランチ戦略の実装
- コードのバックアップとアーカイブ

## 3. 他サービスとの関係

- **CodeBuild**: ソースコードをビルド
- **CodeDeploy**: アプリケーションをデプロイ
- **CodePipeline**: CI/CDパイプラインの構築
- **CloudWatch**: リポジトリのメトリクスとログ
- **IAM**: リポジトリへのアクセス制御
- **CloudTrail**: リポジトリのアクティビティを監査

## 4. 料金のざっくりイメージ

- **ストレージ**: GBあたり$0.06/月（最初の5GB/月は無料）
- **リクエスト**: リクエストあたり$0.001（最初の2,000リクエスト/月は無料）
- **注意**: GitHubやGitLabより高価だが、AWSサービスとの統合が容易

## 5. 初心者がまず覚えるべきポイント

- Gitベースのソースコード管理サービス
- プライベートリポジトリ（GitHubのプライベートリポジトリのようなもの）
- 標準的なGitコマンドで操作可能
- IAMでリポジトリへのアクセスを制御
- ブランチ保護ルールの設定
- プルリクエストとコードレビュー
- CloudWatchでリポジトリのメトリクスを監視
- 暗号化（保存時と転送時）
- AWSサービスとの統合が容易（CodeBuild、CodeDeploy、CodePipeline）
- リージョンごとにリポジトリを作成
