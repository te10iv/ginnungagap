# Amazon Athena：まずこれだけ（Lv1）

## このサービスの一言説明
- Amazon Athena は「**S3データをSQLで分析できる**」AWSサービス

## ゴール（ここまでできたら合格）
- S3データに対して **SQLクエリを実行できる**
- **テーブル定義できる**
- 「S3ログ分析にはAthenaが便利」と判断できる

## まず覚えること（最小セット）
- **サーバーレス**：インフラ管理不要
- **SQL標準**：Presto / Trino ベース
- **S3直接クエリ**：データ移動不要
- **従量課金**：スキャンしたデータ量で課金
- **用途**：ログ分析、データ分析

## できるようになること
- □ S3バケットにテーブル定義できる
- □ SQLクエリを実行できる
- □ CloudTrail / VPC Flow Logs / ALBログを分析できる
- □ クエリ結果をS3に保存できる

## まずやること（Hands-on）
- S3バケット作成、サンプルデータアップロード
- Athenaでデータベース作成
- テーブル定義（CSV/JSON/Parquet）
- SELECT クエリ実行

## 関連するAWSサービス（名前だけ）
- **S3**：データソース
- **Glue**：データカタログ、ETL
- **QuickSight**：可視化
- **CloudTrail / VPC Flow Logs / ALB Logs**：ログ分析
- **Lambda**：クエリ自動化
