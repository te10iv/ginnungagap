# EBS マネージドコンソールでのセットアップ

## 作成するもの

EBSボリュームを作成し、EC2インスタンスにアタッチします。データを永続化するためのストレージを追加します。

## セットアップ手順

1. **EC2コンソールを開く**
   - AWSマネージメントコンソールで「EC2」を検索して選択

2. **ボリュームを作成**
   - 「ボリューム」を選択
   - 「ボリュームを作成」をクリック

3. **ボリューム設定**
   - **サイズ**: 10GBを入力
   - **ボリュームタイプ**: gp3（汎用SSD）を選択
   - **可用性ゾーン**: EC2インスタンスと同じAZを選択
   - **暗号化**: デフォルトの暗号化キーを使用

4. **確認と作成**
   - 設定を確認
   - 「ボリュームを作成」をクリック

5. **EC2インスタンスにアタッチ**
   - 作成したボリュームを選択
   - 「アクション」→「ボリュームをアタッチ」をクリック
   - **インスタンス**: アタッチするEC2インスタンスを選択
   - **デバイス名**: `/dev/sdf` を選択（デフォルト）
   - 「ボリュームをアタッチ」をクリック

6. **ファイルシステムを作成（EC2インスタンス内で実行）**
   - EC2インスタンスにSSH接続
   - 以下のコマンドを実行：
   ```bash
   # ボリュームを確認
   lsblk
   
   # ファイルシステムを作成
   sudo mkfs -t xfs /dev/xvdf
   
   # マウントポイントを作成
   sudo mkdir /mnt/my-volume
   
   # マウント
   sudo mount /dev/xvdf /mnt/my-volume
   ```

## 補足

- ボリュームはEC2インスタンスと同じAZに作成する必要があります
- 再起動後もマウントするには、`/etc/fstab`に設定を追加します
