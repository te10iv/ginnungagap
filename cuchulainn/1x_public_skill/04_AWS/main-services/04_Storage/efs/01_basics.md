# Amazon EFS：まずこれだけ（Lv1）

## このサービスの一言説明
- Amazon EFS は「**複数EC2から同時アクセス可能なファイルストレージ**」を提供するAWSサービス

## ゴール（ここまでできたら合格）
- EFSファイルシステムを **作成できる**
- **複数EC2からマウントできる**
- 「共有ファイルストレージにはEFSが必要」と判断できる

## まず覚えること（最小セット）
- **EFS**：NFSプロトコル、複数EC2から同時アクセス
- **自動スケール**：容量無制限、使用量に応じて課金
- **マウントターゲット**：各AZに配置、EC2からNFSマウント
- **ストレージクラス**：Standard、Infrequent Access（IA）
- **パフォーマンスモード**：汎用、Max I/O

## できるようになること
- □ マネジメントコンソールでEFSファイルシステムを作成できる
- □ マウントターゲットを各AZに配置できる
- □ EC2からNFSマウントできる
- □ 複数EC2から同時アクセスを確認できる

## まずやること（Hands-on）
- EFSファイルシステム作成
- 2つのAZにマウントターゲット配置
- 2台のEC2からNFSマウント
- ファイル作成・共有確認

## 関連するAWSサービス（名前だけ）
- **EC2**：マウント先
- **Lambda**：EFS統合（永続化ストレージ）
- **VPC / Subnet**：マウントターゲット配置
- **Security Group**：アクセス制御
- **Backup**：自動バックアップ
- **DataSync**：オンプレミスからデータ移行
