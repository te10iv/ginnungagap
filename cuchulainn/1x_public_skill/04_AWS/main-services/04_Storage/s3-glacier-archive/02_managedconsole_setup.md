# S3 Glacier マネージドコンソールでのセットアップ

## 作成するもの

S3バケットにライフサイクルポリシーを設定し、古いオブジェクトを自動的にGlacierに移行します。長期アーカイブ用の低コストストレージを設定します。

## セットアップ手順

1. **S3コンソールを開く**
   - AWSマネージメントコンソールで「S3」を検索して選択

2. **バケットを作成（または既存のバケットを使用）**
   - 「バケットを作成」をクリック
   - **バケット名**: `my-archive-bucket-<ランダム文字列>` を入力
   - 基本的な設定を完了

3. **ライフサイクルポリシーを作成**
   - バケットを選択
   - 「管理」タブを選択
   - 「ライフサイクルルールを作成」をクリック

4. **ルール設定**
   - **ルール名**: `glacier-archive-rule` を入力
   - **ルールスコープ**: すべてのオブジェクトに適用

5. **移行アクションを設定**
   - 「移行アクションを追加」をクリック
   - **オブジェクトの作成後**: 30日を入力
   - **移行先**: Glacier Flexible Retrievalを選択

6. **期限切れアクション（オプション）**
   - 必要に応じて期限切れアクションを設定

7. **確認と作成**
   - 設定を確認
   - 「ルールを作成」をクリック

8. **確認**
   - ライフサイクルルールが作成されたことを確認

## 補足

- オブジェクトが30日経過すると自動的にGlacierに移行されます
- Glacierからの取得には時間がかかります（数分〜数時間）
- 取得オプション（Expedited、Standard、Bulk）で取得時間とコストが異なります
