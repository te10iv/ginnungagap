# S3 マネージドコンソールでのセットアップ

## 作成するもの

S3バケットを作成し、静的ウェブサイトをホスティングできるようにします。HTMLファイルをアップロードして公開します。

## セットアップ手順

1. **S3コンソールを開く**
   - AWSマネージメントコンソールで「S3」を検索して選択

2. **バケットを作成**
   - 「バケットを作成」をクリック

3. **一般設定**
   - **バケット名**: `my-website-bucket-<ランダム文字列>` を入力（グローバルで一意である必要がある）
   - **AWSリージョン**: ap-northeast-1を選択

4. **オブジェクト所有権**
   - **ACL**: ACL無効（推奨）を選択

5. **パブリックアクセス設定**
   - **パブリックアクセスをブロック**: すべてのパブリックアクセスをブロックをオフにする（静的ウェブサイトホスティングの場合）
   - 警告を確認してチェックボックスにチェック

6. **バケットバージョニング**
   - **バージョニング**: 無効（デフォルト）

7. **暗号化**
   - **デフォルトの暗号化**: Amazon S3マネージドキー（SSE-S3）を選択

8. **確認と作成**
   - 設定を確認
   - 「バケットを作成」をクリック

9. **静的ウェブサイトホスティングを有効化**
   - バケットを選択
   - 「プロパティ」タブを選択
   - 「静的ウェブサイトホスティング」で「編集」をクリック
   - **静的ウェブサイトホスティング**: 有効にする
   - **インデックスドキュメント**: `index.html` を入力
   - 「変更を保存」をクリック

10. **ファイルをアップロード**
    - 「オブジェクト」タブで「アップロード」をクリック
    - `index.html` ファイルをアップロード
    - 「アップロード」をクリック

## 補足

- バケット名はグローバルで一意である必要があります
- 静的ウェブサイトのURLは「プロパティ」タブの「静的ウェブサイトホスティング」セクションに表示されます
