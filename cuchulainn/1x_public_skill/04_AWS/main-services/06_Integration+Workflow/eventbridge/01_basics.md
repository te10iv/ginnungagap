# Amazon EventBridge：まずこれだけ（Lv1）

## このサービスの一言説明
- Amazon EventBridge は「**イベント駆動型アーキテクチャのイベントバス**」を提供するAWSサービス

## ゴール（ここまでできたら合格）
- イベントルールを **作成できる**
- **Lambda関数をトリガーできる**
- 「イベント駆動にはEventBridgeが必要」と判断できる

## まず覚えること（最小セット）
- **イベントバス**：イベントの配送路
- **ルール**：イベントパターン + ターゲット
- **イベントパターン**：どのイベントを捕捉するか
- **ターゲット**：Lambda、Step Functions、SNS等
- **スケジュール**：cron式による定期実行

## できるようになること
- □ イベントルールを作成できる
- □ スケジュール実行を設定できる（毎日・毎時等）
- □ Lambda関数をターゲット設定できる
- □ カスタムイベントを送信できる

## まずやること（Hands-on）
- ルール作成（スケジュール：毎時0分）
- Lambda関数をターゲット設定
- テスト実行
- カスタムイベント送信

## 関連するAWSサービス（名前だけ）
- **Lambda**：イベントハンドラー
- **Step Functions**：ワークフロー実行
- **SNS / SQS**：イベント配信
- **CloudWatch Logs**：ログ保存
- **API Gateway**：APIイベント
