# EventBridge マネージドコンソールでのセットアップ

## 作成するもの

EventBridgeルールを作成し、スケジュール実行でLambda関数を定期的に実行します。Cron式でスケジュールを設定します。

## セットアップ手順

1. **EventBridgeコンソールを開く**
   - AWSマネージメントコンソールで「EventBridge」を検索して選択

2. **ルールを作成**
   - 「ルール」を選択
   - 「ルールを作成」をクリック

3. **ルールの設定**
   - **名前**: `my-scheduled-rule` を入力
   - **説明**: 任意の説明を入力
   - **イベントバス**: デフォルトを選択

4. **イベントパターンを定義**
   - **イベントソース**: AWSイベントまたはEventBridgeパートナーイベントを選択
   - **イベントパターン**: スケジュールを選択
   - **スケジュール式**: `rate(5 minutes)` または `cron(0 12 * * ? *)` を入力

5. **ターゲットを選択**
   - **ターゲットタイプ**: AWSサービスを選択
   - **ターゲットを選択**: Lambda関数を選択
   - **関数**: 既存のLambda関数を選択

6. **確認と作成**
   - 設定を確認
   - 「次へ」→「ルールを作成」をクリック

7. **確認**
   - ルールが作成され、有効になっていることを確認

## 補足

- スケジュール式で定期的な実行を設定できます
- 複数のターゲット（Lambda、SNS、SQSなど）を設定可能です
