# Amazon SNS：まずこれだけ（Lv1）

## このサービスの一言説明
- Amazon SNS は「**Pub/Sub型メッセージング・通知サービス**」を提供するAWSサービス

## ゴール（ここまでできたら合格）
- SNSトピックを **作成できる**
- **メール通知を設定できる**
- 「通知・メッセージングにはSNSが必要」と判断できる

## まず覚えること（最小セット）
- **トピック**：メッセージ配信先のグループ
- **サブスクリプション**：通知先（メール、SMS、Lambda等）
- **パブリッシュ**：メッセージ送信
- **ファンアウト**：1メッセージを複数宛先に配信
- **プロトコル**：メール、SMS、HTTP、Lambda、SQS等

## できるようになること
- □ トピックを作成できる
- □ メールサブスクリプションを追加できる
- □ メッセージをパブリッシュできる
- □ Lambda統合ができる

## まずやること（Hands-on）
- トピック作成
- メールサブスクリプション追加
- サブスクリプション確認（メール）
- テストメッセージ送信

## 関連するAWSサービス（名前だけ）
- **CloudWatch**：アラーム通知先
- **Lambda**：イベント駆動処理
- **SQS**：メッセージキュー（ファンアウト）
- **EventBridge**：イベントルーティング
- **S3**：イベント通知
