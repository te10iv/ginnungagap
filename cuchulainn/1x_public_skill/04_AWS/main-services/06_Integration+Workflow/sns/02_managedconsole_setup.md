# SNS マネージドコンソールでのセットアップ

## 作成するもの

SNSトピックを作成し、メールアドレスをサブスクライブします。アラーム通知などでメールを送信できるようにします。

## セットアップ手順

1. **SNSコンソールを開く**
   - AWSマネージメントコンソールで「SNS」を検索して選択

2. **トピックを作成**
   - 「トピック」を選択
   - 「トピックを作成」をクリック
   - **名前**: `my-notification-topic` を入力
   - **表示名**: 任意の表示名を入力
   - 「トピックを作成」をクリック

3. **サブスクリプションを作成**
   - 作成したトピックを選択
   - 「サブスクリプションを作成」をクリック
   - **プロトコル**: メールを選択
   - **エンドポイント**: メールアドレスを入力
   - 「サブスクリプションを作成」をクリック

4. **確認メールを承認**
   - 指定したメールアドレスに確認メールが送信されます
   - メール内のリンクをクリックして承認

5. **テストメッセージを送信**
   - トピックを選択
   - 「メッセージを公開」をクリック
   - **メッセージ**: テストメッセージを入力
   - 「メッセージを公開」をクリック

## 補足

- メール以外にもSMS、HTTP、SQS、Lambdaなどにサブスクライブ可能です
- CloudWatchアラームの通知先として使用できます
