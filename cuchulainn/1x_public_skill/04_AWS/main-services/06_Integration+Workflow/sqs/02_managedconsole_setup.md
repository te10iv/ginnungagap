# SQS マネージドコンソールでのセットアップ

## 作成するもの

SQSキューを作成し、メッセージを送受信できるようにします。非同期処理のためのメッセージキューを構築します。

## セットアップ手順

1. **SQSコンソールを開く**
   - AWSマネージメントコンソールで「SQS」を検索して選択

2. **キューを作成**
   - 「キューを作成」をクリック

3. **キュー設定**
   - **キュー名**: `my-queue` を入力
   - **キュータイプ**: 標準キューを選択
   - **可視性タイムアウト**: 30秒（デフォルト）
   - **メッセージ保持期間**: 4日間（デフォルト）

4. **設定を確認**
   - その他の設定はデフォルトのまま
   - 「キューを作成」をクリック

5. **メッセージを送信**
   - 作成したキューを選択
   - 「メッセージを送信」をクリック
   - **メッセージ本文**: テストメッセージを入力
   - 「メッセージを送信」をクリック

6. **メッセージを受信**
   - 「メッセージをポーリング」をクリック
   - メッセージが表示されることを確認

## 補足

- 標準キューとFIFOキューがあります
- Lambda関数をトリガーとして設定可能です
