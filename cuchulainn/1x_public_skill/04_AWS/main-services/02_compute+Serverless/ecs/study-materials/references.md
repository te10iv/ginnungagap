1. **基本概念（タスク、サービス、クラスター）**
   - → AWS Black Belt、公式ドキュメント、AWSコンテナ設計・構築入門書籍

2. **タスク定義（コンテナ定義、リソース割り当て）**
   - → AWSコンテナ設計・構築入門書籍、公式ドキュメント

3. **サービス（レプリカ数、デプロイ戦略）**
   - → AWSコンテナ設計・構築入門書籍、Udemy英語講座

4. **ALB連携**
   - → AWSコンテナ設計・構築入門書籍、公式ドキュメント

5. **Service Connect / Service Discovery**
   - → AWS Black Belt、公式ドキュメント

6. **CodeDeploy連携（Blue/Greenデプロイ）**
   - → AWSコンテナ設計・構築入門書籍、公式ドキュメント

7. **CloudWatch連携（ログ、メトリクス）**
   - → AWSコンテナ設計・構築入門書籍、公式ドキュメント

### おすすめ学習パス

**初学者（ECS初めて、Docker知識あり）**
1. AWS Black Belt動画（ECS、1時間）
2. AWS Hands-on for Beginners（ECS構築、2時間）
3. AWSコンテナ設計・構築入門書籍

**中級者（実務で使いたい）**
1. AWSコンテナ設計・構築入門書籍（全体を読む）
2. 公式ドキュメント（必要箇所）
3. Classmethod記事で実践Tips収集

**上級者（ECSを極める）**
1. Udemy英語講座（ECS & Fargate Masterclass、20時間）
2. 公式ドキュメント（全体を深く）
3. Classmethod記事で最新Tips収集

### ECS on Fargate vs ECS on EC2

| 項目 | ECS on Fargate | ECS on EC2 |
|------|----------------|------------|
| **サーバー管理** | 不要 | 必要 |
| **コスト** | やや高い | 低い |
| **起動速度** | 速い | やや遅い |
| **柔軟性** | 限定的 | 高い |
| **推奨用途** | 小〜中規模 | 大規模・特殊要件 |
