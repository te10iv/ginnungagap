# ECS マネージドコンソールでのセットアップ

## 作成するもの

ECSクラスターとサービスを作成し、Dockerコンテナを実行します。Fargateを使用してサーバーレスでコンテナを実行します。

## セットアップ手順

1. **ECSコンソールを開く**
   - AWSマネージメントコンソールで「ECS」を検索して選択

2. **クラスターを作成**
   - 「クラスターを作成」をクリック
   - **クラスター名**: `my-ecs-cluster` を入力
   - **インフラストラクチャ**: AWS Fargateを選択
   - 「作成」をクリック

3. **タスク定義を作成**
   - 「タスク定義」を選択
   - 「新しいタスク定義の作成」をクリック
   - **タスク定義ファミリー**: `my-task` を入力
   - **起動タイプ**: Fargateを選択
   - **タスクサイズ**: 0.5 vCPU、1 GBメモリを選択
   - **コンテナ**: 「コンテナを追加」をクリック
     - **コンテナ名**: `my-container` を入力
     - **イメージURI**: `nginx:latest` を入力
     - **ポートマッピング**: 80（HTTP）を追加
   - 「作成」をクリック

4. **サービスを作成**
   - クラスターを選択
   - 「サービスを作成」をクリック
   - **起動タイプ**: Fargateを選択
   - **タスク定義**: 作成したタスク定義を選択
   - **サービス名**: `my-service` を入力
   - **希望するタスク数**: 1
   - **VPC**: 作成済みのVPCを選択
   - **サブネット**: パブリックサブネットを選択
   - **セキュリティグループ**: 新しいセキュリティグループを作成（HTTPポート80を許可）
   - 「作成」をクリック

## 補足

- Fargateを使用することで、EC2インスタンスの管理が不要です
- タスクが起動し、実行中になるまで数分かかります
