# Lambda マネージドコンソールでのセットアップ

## 作成するもの

Lambda関数を作成し、S3バケットへのファイルアップロード時に自動実行される関数を設定します。アップロードされたファイルの情報をログに出力します。

## セットアップ手順

1. **Lambdaコンソールを開く**
   - AWSマネージメントコンソールで「Lambda」を検索して選択

2. **関数を作成**
   - 「関数を作成」をクリック
   - **関数名**: `my-lambda-function` を入力
   - **ランタイム**: Python 3.11を選択
   - **アーキテクチャ**: x86_64を選択
   - 「関数を作成」をクリック

3. **関数コードを記述**
   - コードエディタに以下のコードを入力：
   ```python
   import json
   
   def lambda_handler(event, context):
       print("Event: " + json.dumps(event))
       return {
           'statusCode': 200,
           'body': json.dumps('Hello from Lambda!')
       }
   ```
   - 「Deploy」をクリック

4. **トリガーを追加（オプション）**
   - 「トリガーを追加」をクリック
   - **ソース**: S3を選択
   - **バケット**: 既存のS3バケットを選択
   - **イベントタイプ**: All object create eventsを選択
   - 「追加」をクリック

5. **実行ロールの確認**
   - 「設定」タブで「実行ロール」を確認
   - 必要に応じて権限を追加

## 補足

- Lambda関数はイベント駆動で自動実行されます
- CloudWatch Logsにログが出力されます
- トリガーは後から追加・削除可能です
