   - → 公式ドキュメント、Udemy英語講座

### おすすめ学習パス

**初学者（EC2初めて）**
1. AWS Black Belt動画（EC2、1時間）
2. AWS Hands-on for Beginners（EC2起動、2時間）
3. AWSインフラ構築入門書籍

**中級者（実務で使いたい）**
1. Udemy日本語講座（EC2全般、10時間）
2. AWS Black Belt（スポットインスタンス）
3. 公式ドキュメント（必要箇所）
4. Classmethod記事で実践Tips収集

**上級者（EC2を極める）**
1. Udemy英語講座（Nitro System、プレイスメントグループ等、30時間）
2. AWS Well-Architected Framework
3. AWS Well-Architected Labs
4. 公式ドキュメント（全体を深く）

### 実務でよく使う機能ランキング

1. **インスタンス起動・停止** - 最も基本
2. **セキュリティグループ設定** - ほぼすべての環境で使用
3. **AMI作成** - バックアップ、環境複製で頻出
4. **EBS設定** - ストレージ管理で必須
5. **Auto Scaling** - 本番環境で多用
6. **スポットインスタンス** - コスト最適化で重要
7. **プレイスメントグループ** - 高パフォーマンス要件で使用

### インスタンスタイプの選び方

EC2学習で最も重要なのがインスタンスタイプの選び方：

| 用途 | 推奨インスタンスタイプ | 特徴 |
|------|----------------------|------|
| **汎用** | t3, t4g, m5, m6i | バランス型 |
| **コンピューティング最適化** | c5, c6i, c7g | CPU性能重視 |
| **メモリ最適化** | r5, r6i, x2iedn | メモリ容量重視 |
| **ストレージ最適化** | i3, i4i, d3 | ストレージ性能重視 |
| **高速ネットワーク** | m5n, r5n | ネットワーク性能重視 |
| **GPU** | p4, g5 | 機械学習、グラフィックス |
| **Graviton（Arm）** | t4g, m6g, c7g | コスパ重視 |
