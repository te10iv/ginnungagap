# EC2 マネージドコンソールでのセットアップ

## 作成するもの

EC2インスタンスを起動し、基本的なWebサーバーとして使用できるようにします。パブリックIPアドレスを割り当て、インターネットからアクセス可能にします。

## セットアップ手順

1. **EC2コンソールを開く**
   - AWSマネージメントコンソールで「EC2」を検索して選択

2. **インスタンスを起動**
   - 「インスタンスを起動」をクリック
   - **名前**: `my-ec2-instance` を入力

3. **AMIを選択**
   - **AMI**: Amazon Linux 2023を選択（無料利用枠対象）

4. **インスタンスタイプを選択**
   - **インスタンスタイプ**: t3.microを選択（無料利用枠対象）

5. **キーペアを作成**
   - **キーペア**: 新しいキーペアを作成
   - **キーペア名**: `my-key-pair` を入力
   - 「キーペアを作成」をクリックしてダウンロード

6. **ネットワーク設定**
   - **VPC**: 作成済みのVPCを選択
   - **サブネット**: パブリックサブネットを選択
   - **パブリックIPの自動割り当て**: 有効
   - **セキュリティグループ**: 新しいセキュリティグループを作成
     - **名前**: `my-sg` を入力
     - **インバウンドルール**: SSH（ポート22）とHTTP（ポート80）を許可

7. **ストレージを設定**
   - **ボリュームサイズ**: 8GB（デフォルト）

8. **インスタンスを起動**
   - 「インスタンスを起動」をクリック
   - インスタンスの状態が「実行中」になるまで待機

9. **接続**
   - インスタンスを選択して「接続」をクリック
   - SSH接続の手順に従って接続

## 補足

- キーペアファイルは安全に保管してください（再ダウンロード不可）
- セキュリティグループでファイアウォール設定を行います
