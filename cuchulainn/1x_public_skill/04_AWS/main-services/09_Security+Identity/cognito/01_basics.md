# Amazon Cognito：まずこれだけ（Lv1）

## このサービスの一言説明
- Amazon Cognito は「**ユーザー認証・認可のマネージドサービス**」を提供するAWSサービス

## ゴール（ここまでできたら合格）
- ユーザープールを **作成できる**
- **サインアップ・サインインができる**
- 「ユーザー認証にはCognitoが必要」と判断できる

## まず覚えること（最小セット）
- **ユーザープール**：ユーザーディレクトリ（サインアップ・サインイン）
- **アイデンティティプール**：一時的AWS認証情報取得
- **JWT トークン**：IDトークン、アクセストークン、リフレッシュトークン
- **MFA**：多要素認証
- **ホストされたUI**：AWS提供のログイン画面

## できるようになること
- □ ユーザープールを作成できる
- □ ユーザーをサインアップできる
- □ サインインしてトークン取得できる
- □ API GatewayでCognito認証できる

## まずやること（Hands-on）
- ユーザープール作成
- ホストされたUIでサインアップ
- サインイン
- JWTトークン確認

## 関連するAWSサービス（名前だけ）
- **API Gateway**：Cognito Authorizer
- **ALB**：Cognito認証
- **Lambda**：トリガー（Pre SignUp等）
- **IAM**：アイデンティティプール経由
- **S3**：一時的アクセス
