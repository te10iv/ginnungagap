# Cognito マネージドコンソールでのセットアップ

## 作成するもの

Cognitoユーザープールを作成し、ユーザー認証機能を実装します。ユーザーのサインアップ、サインイン機能を提供します。

## セットアップ手順

1. **Cognitoコンソールを開く**
   - AWSマネージメントコンソールで「Cognito」を検索して選択

2. **ユーザープールを作成**
   - 「ユーザープールを作成」をクリック

3. **サインインオプションを設定**
   - **サインインオプション**: ユーザー名とメールアドレスを選択
   - **パスワードポリシー**: デフォルト設定

4. **セキュリティ要件を設定**
   - **多要素認証**: オプション（必要に応じて有効化）

5. **ユーザープール名を設定**
   - **ユーザープール名**: `my-user-pool` を入力

6. **確認と作成**
   - 設定を確認
   - 「ユーザープールを作成」をクリック

7. **アプリクライアントを作成**
   - 作成したユーザープールを選択
   - 「アプリの統合」タブで「アプリクライアントを作成」をクリック
   - **アプリクライアント名**: `my-app-client` を入力
   - 「アプリクライアントを作成」をクリック

8. **ユーザーを作成（テスト用）**
   - 「ユーザーとグループ」タブで「ユーザーを作成」をクリック
   - **ユーザー名**: `testuser` を入力
   - **メールアドレス**: メールアドレスを入力
   - **一時パスワード**: パスワードを設定
   - 「ユーザーを作成」をクリック

## 補足

- ユーザープールIDとアプリクライアントIDをメモしておきます
- アプリケーションからCognito SDKを使用して認証を実装します
