# AWS KMS：まずこれだけ（Lv1）

## このサービスの一言説明
- AWS KMS は「**暗号化キーの作成・管理**」を提供するAWSサービス

## ゴール（ここまでできたら合格）
- KMSキーを **作成できる**
- **データを暗号化・復号化できる**
- 「データ暗号化にはKMSが必要」と判断できる

## まず覚えること（最小セット）
- **CMK（Customer Master Key）**：マスターキー
- **対称キー**：暗号化・復号化に同じキー（デフォルト）
- **非対称キー**：公開鍵・秘密鍵
- **エンベロープ暗号化**：データキーでデータ暗号化、CMKでデータキー暗号化
- **自動ローテーション**：年1回自動ローテーション可能

## できるようになること
- □ KMSキーを作成できる
- □ データを暗号化できる
- □ データを復号化できる
- □ S3/EBS暗号化にKMSキーを指定できる

## まずやること（Hands-on）
- KMSキー作成（対称キー）
- AWS CLIでデータ暗号化
- データ復号化
- S3バケット暗号化設定（KMSキー指定）

## 関連するAWSサービス（名前だけ）
- **S3 / EBS / RDS等**：データ暗号化
- **Secrets Manager**：シークレット暗号化
- **CloudTrail**：KMS操作ログ
- **IAM**：キーアクセス制御
