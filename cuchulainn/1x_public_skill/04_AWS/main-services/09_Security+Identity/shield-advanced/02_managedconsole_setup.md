# Shield Advanced マネージドコンソールでのセットアップ

## 作成するもの

Shield Advancedを有効化し、リソースを保護します。DDoS攻撃からアプリケーションを保護します。

## セットアップ手順

1. **Shieldコンソールを開く**
   - AWSマネージメントコンソールで「Shield」を検索して選択

2. **Shield Advancedを有効化**
   - 「Shield Advancedを有効化」をクリック
   - **サブスクリプション**: 年間契約を選択
   - 利用規約に同意

3. **保護するリソースを選択**
   - **リソースタイプ**: CloudFront、ALB、Route 53などから選択
   - 保護するリソースを選択
   - 「保護を追加」をクリック

4. **通知設定**
   - **SNSトピック**: 通知を受け取るSNSトピックを選択
   - **連絡先**: 緊急連絡先を設定

5. **確認**
   - 保護されているリソースの一覧を確認

## 補足

- Shield Advancedは有料サービスです（年間契約）
- Shield Standard（無料）は自動的に有効になっています
- 24時間365日のDDoS対応チームがサポートします
