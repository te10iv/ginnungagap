# Shield Advanced 概要

## 1. サービスの役割

AWS Shield Advancedは、DDoS攻撃からアプリケーションを保護するマネージドサービスです。Shield Standard（無料）よりも高度な保護機能を提供し、24時間365日のDDoS対応チーム、コスト保護、詳細な攻撃分析などを提供します。

## 2. 主なユースケース

- 大規模なDDoS攻撃からの保護
- エンタープライズレベルのセキュリティ要件
- コンプライアンス要件を満たすためのDDoS保護
- 詳細な攻撃分析とレポート
- カスタム保護ルールの作成
- コスト保護（DDoS攻撃によるスケーリングコストを保護）

## 3. 他サービスとの関係

- **CloudFront**: CloudFrontディストリビューションを保護
- **ALB, NLB**: ロードバランサーを保護
- **Route 53**: DNSを保護
- **EC2, ECS**: EC2インスタンスとECSサービスを保護
- **WAF**: Webアプリケーションファイアウォールと連携
- **CloudWatch**: 攻撃のメトリクスとログ

## 4. 料金のざっくりイメージ

- **Shield Advanced**: リソースあたり$3,000/月（年間契約）
- **データ転送**: 保護されたデータ転送に追加料金なし
- **注意**: Shield Standard（無料）より高価だが、高度な保護機能を提供

## 5. 初心者がまず覚えるべきポイント

- DDoS保護サービス（Shield Standardの上位版）
- **Shield Standard**: 無料、基本的なDDoS保護
- **Shield Advanced**: 有料、高度なDDoS保護とサポート
- 24時間365日のDDoS対応チーム
- コスト保護（DDoS攻撃によるスケーリングコストを保護）
- 詳細な攻撃分析とレポート
- カスタム保護ルールの作成
- リアルタイムの攻撃通知
- エンタープライズレベルのセキュリティ要件に適している
