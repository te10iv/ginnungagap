# IAM マネージドコンソールでのセットアップ

## 作成するもの

IAMユーザーを作成し、S3への読み取り専用アクセス権限を付与します。最小権限の原則に基づいたアクセス制御を設定します。

## セットアップ手順

1. **IAMコンソールを開く**
   - AWSマネージメントコンソールで「IAM」を検索して選択

2. **ユーザーを作成**
   - 「ユーザー」を選択
   - 「ユーザーを追加」をクリック

3. **ユーザー名を設定**
   - **ユーザー名**: `s3-readonly-user` を入力
   - 「次へ」をクリック

4. **権限を設定**
   - **アクセス権限**: 直接アタッチを選択
   - **ポリシー**: `AmazonS3ReadOnlyAccess` を検索して選択
   - 「次へ」をクリック

5. **確認と作成**
   - 設定を確認
   - 「ユーザーを作成」をクリック

6. **アクセスキーを作成（オプション）**
   - 作成したユーザーを選択
   - 「セキュリティ認証情報」タブを選択
   - 「アクセスキーを作成」をクリック
   - アクセスキーIDとシークレットアクセスキーをメモ（再表示不可）

## 補足

- ルートアカウントは使用せず、IAMユーザーを作成して使用することを推奨します
- 最小権限の原則に基づいて権限を付与します
