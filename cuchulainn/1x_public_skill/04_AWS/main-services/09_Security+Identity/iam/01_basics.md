# AWS IAM：まずこれだけ（Lv1）

## このサービスの一言説明
- AWS IAM は「**AWSリソースへのアクセス制御**」を提供するAWSサービス

## ゴール（ここまでできたら合格）
- IAMユーザーを **作成できる**
- **IAMロールをEC2にアタッチできる**
- 「アクセス制御にはIAMが必要」と判断できる

## まず覚えること（最小セット）
- **IAMユーザー**：人間のユーザー
- **IAMロール**：AWSサービス・アプリケーションが使用
- **IAMポリシー**：権限定義（JSON）
- **IAMグループ**：ユーザーのグループ化
- **最小権限の原則**：必要最小限の権限のみ付与

## できるようになること
- □ IAMユーザーを作成できる
- □ IAMポリシーをアタッチできる
- □ IAMロールを作成できる
- □ EC2にIAMロールをアタッチできる

## まずやること（Hands-on）
- IAMユーザー作成（コンソールアクセス有効）
- 管理ポリシーアタッチ（ReadOnlyAccess）
- IAMロール作成（EC2用）
- EC2にIAMロールアタッチ

## 関連するAWSサービス（名前だけ）
- **すべてのAWSサービス**：IAMで認証・認可
- **CloudTrail**：IAM操作ログ
- **Organizations**：マルチアカウント管理
- **Cognito**：エンドユーザー認証
- **SSO**：シングルサインオン
