# AWS Secrets Manager：まずこれだけ（Lv1）

## このサービスの一言説明
- AWS Secrets Manager は「**シークレット（認証情報）の保管・自動ローテーション**」を提供するAWSサービス

## ゴール（ここまでできたら合格）
- シークレットを **作成できる**
- **アプリケーションから取得できる**
- 「パスワード管理にはSecrets Managerが必要」と判断できる

## まず覚えること（最小セット）
- **シークレット**：パスワード、APIキー、トークン等
- **自動ローテーション**：Lambda関数で定期的に更新
- **バージョン管理**：過去のシークレット保持
- **KMS暗号化**：保管時に自動暗号化
- **RDS統合**：RDSパスワード自動ローテーション

## できるようになること
- □ シークレットを作成できる
- □ AWS CLIでシークレットを取得できる
- □ アプリケーションから取得できる
- □ RDSパスワードローテーション設定できる

## まずやること（Hands-on）
- シークレット作成（DBパスワード）
- AWS CLIでシークレット取得
- Lambda関数からシークレット取得
- RDS統合（自動ローテーション設定）

## 関連するAWSサービス（名前だけ）
- **RDS / Aurora**：パスワードローテーション
- **Lambda**：ローテーション関数
- **KMS**：暗号化
- **IAM**：アクセス制御
- **VPC**：プライベート接続
