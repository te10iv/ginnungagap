# Aurora マネージドコンソールでのセットアップ

## 作成するもの

Aurora MySQLクラスターを作成し、高性能なデータベース環境を構築します。自動スケーリングと高可用性を提供します。

## セットアップ手順

1. **RDSコンソールを開く**
   - AWSマネージメントコンソールで「RDS」を検索して選択

2. **データベースを作成**
   - 「データベースを作成」をクリック
   - **データベース作成方法**: 標準作成を選択

3. **エンジンの選択**
   - **エンジンのタイプ**: Amazon Auroraを選択
   - **エディション**: Amazon Aurora MySQL Compatible Editionを選択
   - **バージョン**: 最新バージョンを選択

4. **テンプレートの選択**
   - **テンプレート**: 本番環境を選択

5. **設定**
   - **DBクラスター識別子**: `my-aurora-cluster` を入力
   - **マスターユーザー名**: `admin` を入力
   - **マスターパスワード**: 強力なパスワードを設定

6. **インスタンス構成**
   - **DBインスタンスクラス**: db.t3.mediumを選択

7. **接続**
   - **VPC**: 作成済みのVPCを選択
   - **サブネットグループ**: デフォルトを選択
   - **パブリックアクセス**: いいえ
   - **セキュリティグループ**: 新しいセキュリティグループを作成（MySQLポート3306を許可）

8. **データベース認証**
   - **データベース認証**: パスワード認証を選択

9. **確認と作成**
   - 設定を確認
   - 「データベースを作成」をクリック

## 補足

- Auroraクラスターにはライターインスタンスとリーダーインスタンスが含まれます
- クラスターエンドポイントとリーダーエンドポイントが自動的に作成されます
