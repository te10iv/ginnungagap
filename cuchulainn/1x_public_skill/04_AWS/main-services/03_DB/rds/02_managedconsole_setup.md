# RDS マネージドコンソールでのセットアップ

## 作成するもの

RDSデータベースインスタンス（MySQL）を作成し、アプリケーションから接続できるようにします。基本的なデータベース環境を構築します。

## セットアップ手順

1. **RDSコンソールを開く**
   - AWSマネージメントコンソールで「RDS」を検索して選択

2. **データベースを作成**
   - 「データベースを作成」をクリック
   - **データベース作成方法**: 標準作成を選択

3. **エンジンの選択**
   - **エンジンのタイプ**: MySQLを選択
   - **バージョン**: 最新バージョンを選択

4. **テンプレートの選択**
   - **テンプレート**: 無料利用枠を選択（テスト用）

5. **設定**
   - **DBインスタンス識別子**: `my-rds-instance` を入力
   - **マスターユーザー名**: `admin` を入力
   - **マスターパスワード**: 強力なパスワードを設定

6. **インスタンス構成**
   - **DBインスタンスクラス**: db.t3.microを選択（無料利用枠対象）

7. **ストレージ**
   - **ストレージタイプ**: 汎用SSD（gp3）を選択
   - **割り当て済みストレージ**: 20GB（無料利用枠）

8. **接続**
   - **VPC**: 作成済みのVPCを選択
   - **サブネットグループ**: デフォルトを選択
   - **パブリックアクセス**: いいえ（プライベートサブネットに配置）
   - **セキュリティグループ**: 新しいセキュリティグループを作成（MySQLポート3306を許可）

9. **データベース認証**
   - **データベース認証**: パスワード認証を選択

10. **確認と作成**
    - 設定を確認
    - 「データベースを作成」をクリック

## 補足

- データベースの作成には数分かかります
- エンドポイントをメモしておき、アプリケーションから接続します
