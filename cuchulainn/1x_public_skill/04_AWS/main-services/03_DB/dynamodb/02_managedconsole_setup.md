# DynamoDB マネージドコンソールでのセットアップ

## 作成するもの

DynamoDBテーブルを作成し、NoSQLデータベースとして使用します。ユーザー情報を保存するシンプルなテーブルを構築します。

## セットアップ手順

1. **DynamoDBコンソールを開く**
   - AWSマネージメントコンソールで「DynamoDB」を検索して選択

2. **テーブルを作成**
   - 「テーブルを作成」をクリック

3. **基本設定**
   - **テーブル名**: `Users` を入力
   - **パーティションキー**: `UserId`（文字列）を入力
   - **ソートキー**: なし（シンプルなテーブル）

4. **テーブル設定**
   - **デフォルト設定**: オンデマンドを選択（自動スケーリング）

5. **設定の確認**
   - 設定を確認
   - 「テーブルを作成」をクリック

6. **アイテムを追加（オプション）**
   - テーブルを選択
   - 「アイテムを探索」タブで「アイテムを作成」をクリック
   - **UserId**: `user001` を入力
   - 「属性を追加」で他の属性（Name、Emailなど）を追加
   - 「アイテムを作成」をクリック

## 補足

- オンデマンドモードでは、使用量に応じて自動的にスケーリングされます
- パーティションキーとソートキーでデータを効率的に管理します
