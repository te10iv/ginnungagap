# AWS Client VPN：まずこれだけ（Lv1）

## このサービスの一言説明
- AWS Client VPN は「**リモートユーザーがVPCに安全にアクセスするためのマネージドVPNサービス**」

## ゴール（ここまでできたら合格）
- Client VPNを **作成できる**
- **Site-to-Site VPNとの違いを説明できる**
- 「リモートアクセスにはこれが必要」と判断できる

## まず覚えること（最小セット）
- **Client VPN**：個人ユーザー→VPCへのVPNアクセス
- **エンドポイント**：VPC内のVPNエンドポイント
- **クライアント証明書認証**：相互TLS認証またはActive Directory認証
- **OpenVPN互換**：OpenVPNクライアントで接続可能
- **スプリットトンネル**：VPC向けトラフィックのみVPN経由

## できるようになること
- □ マネジメントコンソールでClient VPNエンドポイントを作成できる
- □ クライアント証明書を生成・配布できる
- □ OpenVPNクライアントで接続できる
- □ 認可ルールを設定できる

## まずやること（Hands-on）
- ACMでサーバー証明書・クライアント証明書を作成
- Client VPNエンドポイントを作成
- ターゲットネットワーク（サブネット）を関連付け
- クライアント設定ファイルをダウンロード
- OpenVPNクライアントで接続確認

## 関連するAWSサービス（名前だけ）
- **VPC**：Client VPNエンドポイントの配置先
- **ACM**：証明書管理（サーバー・クライアント証明書）
- **Active Directory / Cognito**：ユーザー認証
- **CloudWatch Logs**：接続ログ記録
- **Security Group**：VPN経由のアクセス制御
