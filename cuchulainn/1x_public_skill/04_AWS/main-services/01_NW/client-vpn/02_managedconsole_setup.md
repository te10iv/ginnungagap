# Client VPN マネージドコンソールでのセットアップ

## 作成するもの

Client VPNエンドポイントを作成し、個々のクライアントデバイスからVPCに安全に接続できるようにします。リモートワーク環境などで使用します。

## セットアップ手順

1. **Client VPNコンソールを開く**
   - VPCコンソールで「Client VPNエンドポイント」を選択

2. **サーバー証明書を作成（ACM）**
   - ACMコンソールでサーバー証明書をリクエスト（事前に必要）

3. **Client VPNエンドポイントを作成**
   - 「Client VPNエンドポイントを作成」をクリック
   - **名前タグ**: `my-client-vpn` を入力
   - **説明**: 任意の説明を入力
   - **サーバー証明書ARN**: ACMで作成した証明書のARNを入力
   - **認証情報**: 認証方法を選択（Active Directory、SAML、証明書ベースなど）
   - **接続ログ**: CloudWatch Logsに送信するか選択
   - **DNSサーバー**: 必要に応じて設定
   - 「Client VPNエンドポイントを作成」をクリック

4. **ターゲットネットワークを関連付け**
   - 作成したエンドポイントを選択
   - 「ネットワークの関連付け」タブで「ネットワークの関連付け」をクリック
   - **サブネット**: エンドポイントを配置するサブネットを選択
   - **セキュリティグループ**: 適切なセキュリティグループを選択
   - 「関連付け」をクリック

5. **認証ルールを設定**
   - 「認証」タブで「認証ルールを追加」をクリック
   - 認証方法に応じて設定

## 補足

- クライアント証明書の管理が必要です
- 認証にはActive Directory、SAML、証明書ベースの認証が使用可能です
