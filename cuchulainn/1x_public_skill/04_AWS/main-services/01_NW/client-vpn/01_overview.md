# Client VPN 概要

## 1. サービスの役割

AWS Client VPNは、個々のクライアントデバイス（PC、スマートフォンなど）からAWS VPCに安全に接続するためのマネージドVPNサービスです。リモートワークやモバイル環境から、VPC内のリソースにアクセスできます。

## 2. 主なユースケース

- リモートワーク環境からのVPC内リソースへのアクセス
- モバイルデバイスからの安全な接続
- 開発者や運用担当者がVPC内のリソースにアクセス
- 一時的なアクセス権限の付与
- オフィス外からのセキュアな接続

## 3. 他サービスとの関係

- **VPC**: Client VPNエンドポイントをVPCに作成
- **Subnet**: Client VPNエンドポイントを配置するサブネットを指定
- **Security Group**: クライアント接続時のセキュリティ制御
- **Route Table**: クライアントからのトラフィックのルーティング
- **Active Directory / SAML**: 認証に使用
- **EC2, RDS**: Client VPN経由でアクセスするリソース

## 4. 料金のざっくりイメージ

- Client VPNエンドポイント: 約$0.10/時間（約$72/月）
- アクティブなクライアント接続: 約$0.05/時間（約$36/月） × 接続数
- データ転送料金: GBあたり$0.05（アウトバウンド）

## 5. 初心者がまず覚えるべきポイント

- 個々のクライアントデバイスからVPCに接続可能
- 認証にはActive Directory、SAML、証明書ベースの認証が使用可能
- セキュリティグループでクライアントのアクセスを制御
- オンプレミスのVPNサーバーが不要（マネージドサービス）
- 複数のVPCに接続可能（Transit Gateway経由）
- クライアント証明書の管理が必要
