# Route 53 マネージドコンソールでのセットアップ

## 作成するもの

Route 53でホストゾーンを作成し、ドメイン名のDNSレコードを設定します。EC2インスタンスやALBにドメイン名でアクセスできるようにします。

## セットアップ手順

1. **Route 53コンソールを開く**
   - AWSマネージメントコンソールで「Route 53」を検索して選択

2. **ホストゾーンを作成**
   - 「ホストゾーン」を選択
   - 「ホストゾーンを作成」をクリック
   - **ドメイン名**: `example.com` を入力（所有しているドメイン）
   - **タイプ**: パブリックホストゾーンを選択
   - 「ホストゾーンを作成」をクリック

3. **DNSレコードを作成**
   - 作成したホストゾーンを選択
   - 「レコードを作成」をクリック
   - **レコード名**: `www` を入力（www.example.com）
   - **レコードタイプ**: Aレコードを選択
   - **エイリアス**: 有効
   - **エイリアス先**: Application Load Balancerを選択
   - **ALB**: 作成済みのALBを選択
   - 「レコードを作成」をクリック

4. **ネームサーバーを設定**
   - ホストゾーンのネームサーバーをコピー
   - ドメインのレジストラでネームサーバーを更新

## 補足

- パブリックホストゾーンとプライベートホストゾーンがあります
- エイリアスレコードでAWSリソース（ALB、CloudFrontなど）を直接指定可能です
