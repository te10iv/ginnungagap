# CloudFront マネージドコンソールでのセットアップ

## 作成するもの

CloudFrontディストリビューションを作成し、S3バケットのコンテンツを高速配信できるようにします。世界中のエッジロケーションからコンテンツを配信します。

## セットアップ手順

1. **CloudFrontコンソールを開く**
   - AWSマネージメントコンソールで「CloudFront」を検索して選択

2. **ディストリビューションを作成**
   - 「ディストリビューションを作成」をクリック
   - **オリジンドメイン**: S3バケットを選択（またはカスタムオリジン）
   - **名前**: 自動入力される
   - **オリジンアクセス**: オリジンアクセス制御（OAC）またはオリジンアクセスアイデンティティ（OAI）を設定
   - **ビューアープロトコルポリシー**: HTTPとHTTPSを許可（推奨）
   - **許可されたHTTPメソッド**: GET、HEAD（静的コンテンツの場合）
   - **キャッシュキーとオリジンリクエスト**: シンプル（CachingOptimized）を選択
   - **デフォルトルートオブジェクト**: `index.html` を入力
   - **価格クラス**: すべてのエッジロケーションを使用（デフォルト）
   - 「ディストリビューションを作成」をクリック

3. **確認**
   - ディストリビューションの状態が「デプロイ中」から「有効」になるまで待機（数分〜数十分）
   - ドメイン名（例: `d1234567890.cloudfront.net`）をメモ

## 補足

- ディストリビューションのデプロイには時間がかかります
- カスタムドメインとSSL/TLS証明書（ACM）を設定可能です
- キャッシュTTLを設定してオリジンへのリクエストを削減できます
