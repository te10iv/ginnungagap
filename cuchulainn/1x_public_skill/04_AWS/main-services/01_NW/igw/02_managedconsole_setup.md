# Internet Gateway マネージドコンソールでのセットアップ

## 作成するもの

インターネットゲートウェイを作成し、VPCにアタッチします。これにより、パブリックサブネット内のリソースがインターネットにアクセスできるようになります。

## セットアップ手順

1. **インターネットゲートウェイコンソールを開く**
   - VPCコンソールで「インターネットゲートウェイ」を選択

2. **インターネットゲートウェイを作成**
   - 「インターネットゲートウェイを作成」をクリック
   - **名前タグ**: `my-igw` を入力
   - 「インターネットゲートウェイを作成」をクリック

3. **VPCにアタッチ**
   - 作成したインターネットゲートウェイを選択
   - 「アクション」→「VPCにアタッチ」をクリック
   - **VPC**: 作成済みのVPCを選択
   - 「インターネットゲートウェイをアタッチ」をクリック

4. **確認**
   - 状態が「アタッチ済み」になっていることを確認

## 補足

- インターネットゲートウェイはVPCごとに1つしか作成できません
- アタッチ後、ルートテーブルに0.0.0.0/0 → IGWのルートを追加する必要があります
