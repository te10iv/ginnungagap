# NAT Gateway マネージドコンソールでのセットアップ

## 作成するもの

NAT Gatewayを作成し、プライベートサブネットからインターネットへのアウトバウンド接続を可能にします。パブリックサブネットに配置し、プライベートサブネットのルートテーブルに関連付けます。

## セットアップ手順

1. **NAT Gatewayコンソールを開く**
   - VPCコンソールで「NATゲートウェイ」を選択

2. **NAT Gatewayを作成**
   - 「NATゲートウェイを作成」をクリック
   - **名前タグ**: `my-nat-gateway` を入力
   - **サブネット**: パブリックサブネットを選択
   - **Elastic IP割り当て**: 「Elastic IPを割り当てる」をクリックして新しいElastic IPを割り当て
   - 「NATゲートウェイを作成」をクリック

3. **ルートテーブルに関連付け**
   - プライベートサブネットのルートテーブルを選択
   - 「ルート」タブで「ルートを編集」をクリック
   - 「ルートを追加」をクリック
   - **送信先**: `0.0.0.0/0` を入力
   - **ターゲット**: 作成したNAT Gatewayを選択
   - 「変更を保存」をクリック

4. **確認**
   - NAT Gatewayの状態が「利用可能」になっていることを確認

## 補足

- NAT Gatewayはパブリックサブネットに配置する必要があります
- Elastic IPが自動的に割り当てられます
- 高可用性のため、各AZにNAT Gatewayを配置することを推奨します
