# VPC マネージドコンソールでのセットアップ

## 作成するもの

基本的なVPCを作成します。プライベートなネットワーク環境を構築し、EC2インスタンスなどのリソースを配置できるようにします。

## セットアップ手順

1. **VPCコンソールを開く**
   - AWSマネージメントコンソールで「VPC」を検索して選択

2. **VPCを作成**
   - 「VPCを作成」をクリック
   - **名前タグ**: `my-vpc` を入力
   - **IPv4 CIDRブロック**: `10.0.0.0/16` を入力
   - **IPv6 CIDRブロック**: なし（デフォルト）
   - **テナンシー**: デフォルト
   - 「VPCを作成」をクリック

3. **確認**
   - 作成されたVPCが一覧に表示されることを確認
   - VPC IDをメモしておく

## 補足

- このVPCにはまだサブネットやインターネットゲートウェイがないため、インターネットアクセスはできません
- 次にサブネットとインターネットゲートウェイを作成する必要があります
