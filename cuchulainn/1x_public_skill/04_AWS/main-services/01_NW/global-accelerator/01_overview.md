# Global Accelerator 概要

## 1. サービスの役割

AWS Global Acceleratorは、グローバルなアプリケーションのパフォーマンスと可用性を向上させるサービスです。AWSのグローバルネットワークインフラを使用して、ユーザーから最も近いAWSリージョンのエンドポイントに自動的にルーティングします。

## 2. 主なユースケース

- グローバルなアプリケーションのレイテンシー改善
- 複数リージョンでの高可用性アーキテクチャ
- エンドポイントの自動フェイルオーバー
- 固定IPアドレスが必要なアプリケーション
- DDoS攻撃の緩和（AWS Shield Advancedと連携）
- グローバルなロードバランシング

## 3. 他サービスとの関係

- **ALB, NLB, EC2**: エンドポイントとして設定
- **Route 53**: DNS設定と連携
- **CloudFront**: 静的コンテンツはCloudFront、動的コンテンツはGlobal Acceleratorを使い分け
- **AWS Shield**: DDoS保護と連携
- **VPC**: VPC内のリソースをエンドポイントとして設定

## 4. 料金のざっくりイメージ

- 固定料金: 約$1.50/日（約$45/月）
- データ転送: GBあたり$0.01（最初の10TBまで）
- **注意**: CloudFrontより高価だが、動的コンテンツに適している

## 5. 初心者がまず覚えるべきポイント

- CloudFrontは静的コンテンツ向け、Global Acceleratorは動的コンテンツ向け
- 固定IPアドレスを提供（ホワイトリスト登録などに便利）
- 自動的に最適なエンドポイントにルーティング（ヘルスチェックベース）
- 複数リージョンでの自動フェイルオーバー
- エンドポイントとしてALB、NLB、EC2インスタンス、VPC内のリソースを設定可能
- トラフィックの分散（重み付け）が可能
