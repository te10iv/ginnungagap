# Global Accelerator マネージドコンソールでのセットアップ

## 作成するもの

Global Acceleratorを作成し、複数リージョンのエンドポイント（ALB、NLB、EC2）にグローバルにアクセスできるようにします。固定IPアドレスと自動フェイルオーバーを提供します。

## セットアップ手順

1. **Global Acceleratorコンソールを開く**
   - AWSマネージメントコンソールで「Global Accelerator」を検索して選択

2. **アクセラレーターを作成**
   - 「アクセラレーターを作成」をクリック
   - **名前**: `my-accelerator` を入力
   - **IPアドレスタイプ**: IPv4を選択
   - **プロトコル**: TCP、UDP、または両方を選択
   - **ポート**: デフォルトポート（80、443）またはカスタムポートを選択
   - 「次へ」をクリック

3. **リスナーを設定**
   - **プロトコル**: TCPを選択
   - **ポート範囲**: `80` から `80` を入力
   - 「次へ」をクリック

4. **エンドポイントグループを追加**
   - **リージョン**: エンドポイントがあるリージョンを選択
   - **トラフィックダイヤル**: 100%（デフォルト）
   - **ヘルスチェック**: 有効
   - **エンドポイント**: 「エンドポイントを追加」をクリック
     - **エンドポイントタイプ**: Application Load Balancerを選択
     - **エンドポイント**: ALBを選択
     - **重み**: 100（デフォルト）
   - 「次へ」をクリック

5. **確認と作成**
   - 設定を確認
   - 「アクセラレーターを作成」をクリック

6. **確認**
   - アクセラレーターのIPアドレス（2つの固定IP）をメモ

## 補足

- アクセラレーターには2つの固定IPアドレスが割り当てられます
- 複数のリージョンにエンドポイントを追加可能です
- ヘルスチェックに基づいて自動的に最適なエンドポイントにルーティングされます
