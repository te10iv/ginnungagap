# サブネット マネージドコンソールでのセットアップ

## 作成するもの

VPC内にパブリックサブネットとプライベートサブネットを作成します。パブリックサブネットにはインターネットアクセス可能なリソース、プライベートサブネットにはインターネットから直接アクセスできないリソースを配置できます。

## セットアップ手順

### パブリックサブネットの作成

1. **サブネットコンソールを開く**
   - VPCコンソールで「サブネット」を選択

2. **サブネットを作成**
   - 「サブネットを作成」をクリック
   - **VPC**: 作成済みのVPCを選択
   - **名前タグ**: `public-subnet-1a` を入力
   - **アベイラビリティゾーン**: `ap-northeast-1a` を選択
   - **IPv4 CIDRブロック**: `10.0.1.0/24` を入力
   - 「サブネットを作成」をクリック

### プライベートサブネットの作成

3. **プライベートサブネットを作成**
   - 再度「サブネットを作成」をクリック
   - **VPC**: 同じVPCを選択
   - **名前タグ**: `private-subnet-1a` を入力
   - **アベイラビリティゾーン**: `ap-northeast-1a` を選択
   - **IPv4 CIDRブロック**: `10.0.2.0/24` を入力
   - 「サブネットを作成」をクリック

## 補足

- パブリックサブネットをインターネットアクセス可能にするには、インターネットゲートウェイとルートテーブルの設定が必要です
- プライベートサブネットからインターネットにアクセスするには、NAT Gatewayが必要です
