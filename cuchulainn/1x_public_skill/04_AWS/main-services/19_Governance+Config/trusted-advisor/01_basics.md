# AWS Trusted Advisor：まずこれだけ（Lv1）

## このサービスの一言説明
- AWS Trusted Advisor は「**AWSベストプラクティスチェック**」を提供するAWSサービス

## ゴール（ここまでできたら合格）
- Trusted Advisor ダッシュボードを **確認できる**
- **5つのカテゴリを理解できる**
- 「コスト最適化・セキュリティにTrusted Advisorが役立つ」と判断できる

## まず覚えること（最小セット）
- **5つのカテゴリ**：
  1. コスト最適化
  2. セキュリティ
  3. 耐障害性
  4. パフォーマンス
  5. サービスクォータ
- **チェック結果**：OK / 推奨事項 / 調査推奨
- **無料チェック**：Basic・Developer：7項目
- **有料チェック**：Business・Enterprise：全項目（115+）

## できるようになること
- □ ダッシュボード確認できる
- □ 推奨事項理解できる
- □ コスト最適化チェック活用できる

## まずやること（Hands-on）
- Trusted Advisor コンソール確認
- 無料チェック結果確認
- 推奨事項対応

## 関連するAWSサービス（名前だけ）
- **CloudWatch Events**：チェック結果通知
- **Cost Explorer**：コスト分析
- **Config**：コンプライアンス
- **Security Hub**：セキュリティ統合
