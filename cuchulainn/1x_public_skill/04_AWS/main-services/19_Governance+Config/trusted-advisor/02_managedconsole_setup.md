# Trusted Advisor マネージドコンソールでのセットアップ

## 作成するもの

Trusted Advisorを確認し、AWS環境のベストプラクティスに基づく推奨事項を確認します。コスト最適化とセキュリティの推奨事項を確認します。

## セットアップ手順

1. **Trusted Advisorコンソールを開く**
   - AWSマネージメントコンソールで「Trusted Advisor」を検索して選択

2. **推奨事項を確認**
   - ダッシュボードで5つのカテゴリの推奨事項を確認：
     - **コスト最適化**: 未使用のリソース、Reserved Instanceの推奨など
     - **セキュリティ**: セキュリティグループの設定、IAMのベストプラクティスなど
     - **パフォーマンス**: プロビジョンドIOPSの最適化など
     - **フォールトトレランス**: マルチAZ配置、バックアップなど
     - **サービス制限**: リソースの使用率が制限に近づいている場合の警告

3. **推奨事項を実装**
   - 各推奨事項をクリックして詳細を確認
   - 必要に応じて推奨事項に基づいてリソースを最適化

4. **アラートを設定（Business/Enterpriseサポートプランの場合）**
   - 「環境設定」でアラートを設定
   - SNSトピックを選択して通知を受け取る

## 補足

- Trusted Advisor Basic（無料）では7つのコアチェックが利用可能です
- Business/Enterpriseサポートプランでは全チェックが利用可能です
