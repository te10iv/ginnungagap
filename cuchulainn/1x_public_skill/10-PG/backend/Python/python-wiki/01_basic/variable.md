# 変数

## 1. 要点まとめ

- 変数は値を格納するための名前付きの場所
- Pythonでは変数の宣言と代入が同時に行われる
- 変数名は英数字とアンダースコア（`_`）で構成され、数字で始められない
- 変数名は大文字小文字を区別する（case-sensitive）
- 予約語（キーワード）は変数名に使えない
- 変数の型は動的に決定される（動的型付け）

## 2. 詳細解説

Pythonでは、変数は値を格納するための名前付きの場所です。変数を使用するには、まず変数に値を代入する必要があります。`変数名 = 値`という形式で代入を行います。

変数の宣言と代入は同時に行われます。つまり、変数を宣言するための特別なキーワード（`var`, `let`, `const`など）は必要ありません。変数に値を代入すると、その変数が自動的に作成されます。

変数名には英数字とアンダースコア（`_`）を使用できますが、数字で始めることはできません。また、大文字と小文字は区別されるため、`name`と`Name`は別の変数として扱われます。

Pythonには予約語（キーワード）があり、これらは変数名として使用できません。`if`, `for`, `def`, `class`, `import`などが予約語です。

Pythonは動的型付け言語のため、変数の型は代入される値によって自動的に決定されます。同じ変数に異なる型の値を代入することも可能ですが、コードの可読性を考えると避けるべきです。

## 3. サンプルコード

```python
# 変数の代入
name = "山田太郎"
age = 25
height = 175.5
is_student = True

# 変数の再代入
age = 26  # 年齢を更新
print(age)  # 26

# 変数の型は動的に決定される
x = 10
print(type(x))  # <class 'int'>

x = "文字列"
print(type(x))  # <class 'str'>

# 複数の変数に同時に代入
a, b, c = 1, 2, 3
print(a, b, c)  # 1 2 3

# 変数の値の交換
x, y = 10, 20
x, y = y, x  # 値の交換
print(x, y)  # 20 10

# 変数名の規則
user_name = "太郎"  # アンダースコア使用OK
userName = "花子"  # キャメルケースもOK（ただしPythonではスネークケースが推奨）
_user = "内部変数"  # アンダースコアで始まる変数（慣習的に内部使用）
```

## 4. 注意点・落とし穴

- **未定義変数の使用**: 値を代入する前に変数を使用すると`NameError`が発生します
- **予約語の使用**: `if`, `for`, `def`などを変数名に使うと`SyntaxError`になります
- **型の混在**: 同じ変数に異なる型の値を代入すると、コードが読みにくくなります
- **グローバル変数とローカル変数**: 関数内でグローバル変数を変更する場合は`global`キーワードが必要です
- **変数名の命名規則**: PEP 8では、変数名は小文字とアンダースコアを使用するスネークケースが推奨されています

## 5. 演習問題

1. 変数`name`に自分の名前を、`age`に年齢を代入し、それらを使って自己紹介文を表示してください
2. 変数`x`と`y`に数値を代入し、それらの値を交換してください
3. 複数の変数に同時に値を代入する方法を使って、`a`, `b`, `c`にそれぞれ1, 2, 3を代入してください

## 6. 解答例

```python
# 1. 自己紹介
name = "山田太郎"
age = 25
print(f"私の名前は{name}です。{age}歳です。")

# 2. 値の交換
x = 10
y = 20
print(f"交換前: x={x}, y={y}")
x, y = y, x
print(f"交換後: x={x}, y={y}")

# 3. 複数変数への同時代入
a, b, c = 1, 2, 3
print(f"a={a}, b={b}, c={c}")
```
