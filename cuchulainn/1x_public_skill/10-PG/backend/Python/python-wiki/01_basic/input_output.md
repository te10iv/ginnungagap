# 入力と出力

## 1. 要点まとめ

- `print()`関数で値を出力（表示）する
- `input()`関数でユーザーからの入力を受け取る
- `input()`は常に文字列（str型）を返す
- f-string（f"..."）を使うと変数を文字列に埋め込める
- 複数の値を`print()`で出力する場合はカンマで区切る
- `print()`の`sep`と`end`パラメータで区切り文字と末尾文字を変更できる

## 2. 詳細解説

Pythonでの入出力は、主に`print()`関数と`input()`関数を使用します。

`print()`関数は、引数として渡された値を標準出力（通常はコンソール）に表示します。複数の値を渡すと、デフォルトではスペースで区切られて表示されます。`sep`パラメータで区切り文字を、`end`パラメータで末尾の文字（デフォルトは改行`\n`）を変更できます。

`input()`関数は、ユーザーからの入力を受け取ります。引数としてプロンプト文字列を渡すと、それが表示されてから入力待ちになります。`input()`は常に文字列（str型）を返すため、数値として扱う場合は`int()`や`float()`で変換する必要があります。

文字列に変数の値を埋め込む方法は複数ありますが、Python 3.6以降ではf-string（フォーマット済み文字列リテラル）が推奨されます。`f"文字列{変数名}"`の形式で、変数の値を文字列に埋め込めます。

## 3. サンプルコード

```python
# print()の基本
print("Hello, World!")
print(123)
print(3.14)

# 複数の値を出力
print("山田", "太郎", 25)  # 山田 太郎 25（スペース区切り）

# sepパラメータで区切り文字を変更
print("山田", "太郎", 25, sep="-")  # 山田-太郎-25

# endパラメータで末尾文字を変更
print("Hello", end="")
print("World")  # HelloWorld（改行なし）

# input()の基本
name = input("名前を入力してください: ")
print(f"こんにちは、{name}さん")

# 数値の入力（文字列から変換）
age_str = input("年齢を入力してください: ")
age = int(age_str)  # 文字列を整数に変換
print(f"あなたは{age}歳です")

# f-stringの使用
name = "山田太郎"
age = 25
height = 175.5
print(f"名前: {name}, 年齢: {age}, 身長: {height}cm")

# 式も埋め込める
x = 10
y = 20
print(f"{x} + {y} = {x + y}")  # 10 + 20 = 30
```

## 4. 注意点・落とし穴

- **input()の戻り値は文字列**: `input()`は常に文字列を返すため、数値として扱う場合は`int()`や`float()`で変換が必要です。変換できない値が入力されると`ValueError`が発生します
- **改行の扱い**: `print()`はデフォルトで末尾に改行を出力します。改行を抑制したい場合は`end=""`を指定します
- **f-stringの波括弧**: f-string内で波括弧`{}`を文字として表示したい場合は、`{{`と`}}`のように2つ重ねます
- **エラーハンドリング**: `input()`で数値変換する際は、`try-except`でエラーハンドリングを行うのが推奨されます
- **文字列の連結**: 文字列と数値を`+`で連結しようとすると`TypeError`になります。f-stringを使うか、`str()`で変換しましょう

## 5. 演習問題

1. ユーザーに名前を入力させ、「こんにちは、[名前]さん」と表示するプログラムを作成してください
2. ユーザーに2つの数値を入力させ、それらの和を計算して表示するプログラムを作成してください
3. f-stringを使って、変数`name`、`age`、`city`の値を「私は[名前]です。[年齢]歳で、[都市]に住んでいます」という形式で表示してください

## 6. 解答例

```python
# 1. 名前の入力と表示
name = input("名前を入力してください: ")
print(f"こんにちは、{name}さん")

# 2. 2つの数値の和
num1_str = input("1つ目の数値を入力してください: ")
num2_str = input("2つ目の数値を入力してください: ")
num1 = int(num1_str)
num2 = int(num2_str)
result = num1 + num2
print(f"{num1} + {num2} = {result}")

# 3. f-stringでの表示
name = "山田太郎"
age = 25
city = "東京"
print(f"私は{name}です。{age}歳で、{city}に住んでいます")
```
