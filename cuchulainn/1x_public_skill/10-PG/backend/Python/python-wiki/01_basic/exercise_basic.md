# 基本編 演習

## 1. 要点まとめ

- 変数の宣言と代入
- データ型の理解と型変換
- 演算子の使用
- 入力と出力の操作
- 基本的な計算処理

## 2. 詳細解説

この演習では、Pythonの基本文法を理解するための問題を解きます。変数の使い方、データ型、演算子、入出力など、これまで学んだ内容を総合的に使って問題を解決します。

各問題は段階的に難易度が上がるように設計されています。最初は簡単な計算から始まり、徐々に複雑な処理へと進みます。

## 3. サンプルコード

```python
# 演習問題の例
# 問題1: 変数を使って計算
x = 10
y = 5
result = x + y * 2
print(result)  # 20

# 問題2: 型変換
num_str = "100"
num_int = int(num_str)
print(num_int * 2)  # 200
```

## 4. 注意点・落とし穴

- 変数の型を意識する（特に文字列と数値の違い）
- 演算子の優先順位に注意する
- `input()`の戻り値は文字列であることを忘れない
- エラーメッセージを読んで、どこが間違っているか理解する

## 5. 演習問題

### 問題1: 基本計算
変数`a`に10、`b`に3を代入し、以下の計算結果を表示してください：
- `a + b`
- `a - b`
- `a * b`
- `a / b`
- `a // b`（整数除算）
- `a % b`（剰余）

### 問題2: 型変換
文字列`"50"`を整数に変換し、それを2倍した値を表示してください。

### 問題3: ユーザー入力
ユーザーに半径を入力させ、円の面積を計算して表示するプログラムを作成してください。
（円の面積 = 半径 × 半径 × 3.14）

### 問題4: 複合演算
変数`x`に100を代入し、以下の操作を順番に実行してください：
1. `x`に10を加える
2. `x`を2倍する
3. `x`から5を引く
4. 最終的な`x`の値を表示する

### 問題5: 文字列操作
変数`first_name`に"太郎"、`last_name`に"山田"を代入し、以下の形式で表示してください：
- "山田 太郎"
- "山田太郎"
- "Taro Yamada"（ローマ字表記）

## 6. 解答例

```python
# 問題1: 基本計算
a = 10
b = 3
print(f"a + b = {a + b}")  # 13
print(f"a - b = {a - b}")  # 7
print(f"a * b = {a * b}")  # 30
print(f"a / b = {a / b}")  # 3.333...
print(f"a // b = {a // b}")  # 3
print(f"a % b = {a % b}")  # 1

# 問題2: 型変換
num_str = "50"
num_int = int(num_str)
result = num_int * 2
print(result)  # 100

# 問題3: 円の面積計算
radius_str = input("半径を入力してください: ")
radius = float(radius_str)
area = radius * radius * 3.14
print(f"円の面積は{area}です")

# 問題4: 複合演算
x = 100
x += 10  # x = x + 10
x *= 2   # x = x * 2
x -= 5   # x = x - 5
print(f"最終的なxの値: {x}")  # 215

# 問題5: 文字列操作
first_name = "太郎"
last_name = "山田"
print(f"{last_name} {first_name}")  # 山田 太郎
print(f"{last_name}{first_name}")  # 山田太郎
print("Taro Yamada")  # ローマ字表記（実際の変換は複雑なので簡略化）
```
