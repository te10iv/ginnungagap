# 基本文法

## 1. 要点まとめ

- Pythonの文は改行で区切られる（セミコロン不要）
- コメントは`#`で始まる
- 文字列は`'`（シングルクォート）または`"`（ダブルクォート）で囲む
- インデント（字下げ）がブロック構造を表す
- 変数名は大文字小文字を区別する（case-sensitive）
- 予約語（`if`, `for`, `def`など）は変数名に使えない

## 2. 詳細解説

Pythonの基本文法は非常にシンプルです。文の終わりにセミコロン（`;`）を付ける必要はなく、改行が文の区切りとなります。ただし、1行に複数の文を書く場合はセミコロンで区切ることができます。

コメントは`#`から行末までが無視されます。複数行のコメントは`"""`（トリプルクォート）を使うか、各行に`#`を付けます。

文字列リテラルは`'`または`"`で囲みます。どちらを使っても同じですが、文字列内に`'`を含めたい場合は`"`で囲み、`"`を含めたい場合は`'`で囲むと便利です。

Pythonの最大の特徴は、インデント（字下げ）が構文の一部であることです。同じインデントレベルのコードが同じブロックとして扱われます。通常はスペース4つを使用します（PEP 8の推奨）。

## 3. サンプルコード

```python
# コメントの例
# これは1行コメントです

"""
これは複数行コメントです
複数行にわたって説明を書けます
"""

# 文字列の例
message1 = 'シングルクォートで囲む'
message2 = "ダブルクォートで囲む"
message3 = "It's a beautiful day"  # 文字列内に'を含む場合
message4 = '彼は"こんにちは"と言った'  # 文字列内に"を含む場合

# インデントによるブロック構造
x = 10
if x > 5:
    print("xは5より大きい")  # この行はifブロック内
    print("この行もifブロック内")  # 同じインデントレベル
print("この行はifブロックの外")  # インデントが戻っている

# 変数名の大文字小文字の区別
name = "太郎"
Name = "花子"  # nameとNameは別の変数
print(name)  # "太郎"が表示される
print(Name)  # "花子"が表示される
```

## 4. 注意点・落とし穴

- **インデントの混在**: スペースとタブを混在させるとエラーになります。エディタの設定で「タブをスペースに変換」を有効にしましょう
- **予約語の使用**: `if`, `for`, `def`, `class`などは変数名に使えません。エラーになります
- **文字列のクォート**: 開始と終了のクォートが一致していないと`SyntaxError`になります
- **長い行の改行**: 1行が長すぎる場合は`\`で改行できますが、括弧内では自動的に継続されます

## 5. 演習問題

1. コメントを使って、自分の名前と年齢を説明するプログラムを作成してください
2. 文字列内に`'`と`"`の両方を含む文字列を作成してください
3. インデントを使って、ネストした条件分岐（if文の中にif文）を作成してください

## 6. 解答例

```python
# 1. コメントの使用
# 名前: 山田太郎
# 年齢: 25歳
name = "山田太郎"
age = 25
print(f"{name}、{age}歳")

# 2. 文字列内に'と"を含む
message = '彼は"It\'s a beautiful day"と言った'
# または
message = "彼は\"It's a beautiful day\"と言った"
# または
message = """彼は"It's a beautiful day"と言った"""
print(message)

# 3. ネストした条件分岐
score = 85
if score >= 80:
    print("優秀です")
    if score >= 90:
        print("特に優秀です")
    else:
        print("よくできました")
else:
    print("もう少し頑張りましょう")
```
