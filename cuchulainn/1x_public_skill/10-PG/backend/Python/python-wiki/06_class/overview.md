# 概要

## 1. 要点まとめ

- クラスはオブジェクトの設計図（テンプレート）
- インスタンスはクラスから作成された具体的なオブジェクト
- `__init__`メソッドで初期化処理を行う
- メソッドはクラス内で定義された関数
- 属性（アトリビュート）はオブジェクトが持つデータ
- クラスを使うことでコードの再利用性と保守性が向上する

## 2. 詳細解説

クラスは、オブジェクト指向プログラミングの基本概念です。クラスはオブジェクトの設計図（テンプレート）で、同じ構造と動作を持つオブジェクトを複数作成できます。

クラスから作成された具体的なオブジェクトをインスタンスと呼びます。`クラス名()`のように呼び出すことで、新しいインスタンスを作成できます。

`__init__`メソッドは、インスタンスが作成される際に自動的に呼び出される特殊なメソッドです。初期化処理（属性の設定など）を行います。

メソッドは、クラス内で定義された関数です。メソッドの第1引数は通常`self`で、インスタンス自身を表します。

属性（アトリビュート）は、オブジェクトが持つデータです。`self.属性名`の形式でアクセスします。

クラスを使うことで、関連するデータと処理をまとめて管理でき、コードの再利用性と保守性が向上します。

## 3. サンプルコード

```python
# 基本的なクラス定義
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def introduce(self):
        print(f"私は{self.name}です。{self.age}歳です")

# インスタンスの作成
person1 = Person("太郎", 25)
person1.introduce()  # "私は太郎です。25歳です"

person2 = Person("花子", 30)
person2.introduce()  # "私は花子です。30歳です"
```

## 4. 注意点・落とし穴

- `self`はメソッドの第1引数として必須
- `__init__`は初期化メソッドで、戻り値は返さない
- 属性は`self.属性名`でアクセスする
