# 概要

## 1. 要点まとめ

- モジュールは関数やクラスをまとめたファイル
- `import`文でモジュールを読み込む
- 標準ライブラリはPythonに標準で含まれるモジュール群
- サードパーティライブラリは`pip`でインストールする
- モジュールを使うことでコードを再利用できる
- `from ... import ...`で特定の要素だけをインポートできる

## 2. 詳細解説

モジュールは、関数やクラス、変数などをまとめたファイルです。モジュールを使うことで、コードを再利用可能な単位に分割し、大規模なプログラムを管理しやすくします。

Pythonには標準ライブラリと呼ばれる、よく使われる機能を提供するモジュール群が標準で含まれています。`math`（数学関数）、`datetime`（日時処理）、`os`（OS機能）、`json`（JSON処理）などが代表例です。

モジュールを読み込むには`import`文を使います。`import モジュール名`の形式で、モジュール全体を読み込みます。特定の要素だけを読み込む場合は、`from モジュール名 import 要素名`の形式を使います。

サードパーティライブラリ（外部ライブラリ）を使う場合は、`pip`コマンドでインストールする必要があります。`pip install パッケージ名`でインストールできます。

## 3. サンプルコード

```python
# モジュールのインポート
import math
print(math.pi)  # 3.14159...

# 特定の要素だけをインポート
from math import sqrt
print(sqrt(16))  # 4.0

# エイリアス（別名）を付ける
import datetime as dt
now = dt.datetime.now()
```

## 4. 注意点・落とし穴

- モジュール名と変数名が衝突しないようにする
- `from ... import *`は避ける（名前空間が汚染される）
- 標準ライブラリの存在を知っておく

## 5. 演習問題

1. `math`モジュールをインポートし、円周率を表示してください
2. `datetime`モジュールを使って、現在の日時を表示してください
3. `random`モジュールを使って、1から10までの乱数を生成してください

## 6. 解答例

```python
# 1. mathモジュール
import math
print(math.pi)

# 2. datetimeモジュール
import datetime
now = datetime.datetime.now()
print(now)

# 3. randomモジュール
import random
number = random.randint(1, 10)
print(number)
```
