# if

## 1. 要点まとめ

- `if`文で条件分岐を行う
- `elif`で複数の条件を分岐できる
- `else`でそれ以外の場合を処理できる
- 条件式は真偽値（`True`/`False`）を返す必要がある
- インデントでブロックを表現する
- 条件式には比較演算子や論理演算子が使える

## 2. 詳細解説

`if`文は、条件が`True`のときに特定の処理を実行するための構文です。Pythonでは、`if`の後に条件式を書き、その条件が`True`の場合に実行されるコードをインデントして記述します。

複数の条件を分岐させたい場合は、`elif`（else ifの略）を使用します。`elif`は前の条件が`False`の場合に評価され、`True`の場合はそのブロックが実行されます。

すべての条件が`False`の場合に実行する処理は`else`で指定します。`else`は条件式を持たず、前のすべての条件が`False`の場合に実行されます。

条件式には比較演算子（`==`, `!=`, `<`, `>`, `<=`, `>=`）や論理演算子（`and`, `or`, `not`）を使用できます。また、値そのものが真偽値として評価される場合もあります（0、空文字列、空リストなどは`False`として評価されます）。

## 3. サンプルコード

```python
# 基本的なif文
age = 20
if age >= 18:
    print("成人です")

# if-else文
score = 85
if score >= 60:
    print("合格です")
else:
    print("不合格です")

# if-elif-else文
temperature = 25
if temperature > 30:
    print("暑いです")
elif temperature > 20:
    print("快適です")
elif temperature > 10:
    print("少し寒いです")
else:
    print("寒いです")

# 論理演算子を使った条件
age = 25
has_license = True
if age >= 18 and has_license:
    print("運転できます")

# ネストしたif文
x = 10
y = 20
if x > 0:
    if y > 0:
        print("xとyは両方とも正の数です")
    else:
        print("xは正だが、yは正ではない")
else:
    print("xは正の数ではない")
```

## 4. 注意点・落とし穴

- **`==`と`=`の混同**: 条件式では`==`（等価）を使います。`=`は代入演算子なので、条件式で使うと`SyntaxError`になります
- **インデントエラー**: `if`文のブロックは必ずインデントする必要があります。インデントがないと`IndentationError`になります
- **Noneの比較**: `None`との比較は`==`ではなく`is None`を使うのが推奨されます
- **真偽値の評価**: 数値の`0`、空文字列`""`、空リスト`[]`などは`False`として評価されます。意図しない動作に注意しましょう
- **複数の条件**: `and`と`or`の優先順位に注意が必要です。括弧を使って明示的に指定するのが安全です

## 5. 演習問題

1. 変数`score`に点数を代入し、90点以上なら「優秀」、70点以上なら「良好」、60点以上なら「可」、それ以下なら「不可」と表示するプログラムを作成してください
2. 変数`year`に年を代入し、うるう年かどうかを判定するプログラムを作成してください（うるう年は4で割り切れる年。ただし100で割り切れる年はうるう年ではない。ただし400で割り切れる年はうるう年）
3. 変数`age`と`has_license`を使って、18歳以上かつ免許を持っている場合のみ「運転できます」と表示するプログラムを作成してください

## 6. 解答例

```python
# 1. 成績判定
score = 85
if score >= 90:
    print("優秀")
elif score >= 70:
    print("良好")
elif score >= 60:
    print("可")
else:
    print("不可")

# 2. うるう年判定
year = 2024
if year % 400 == 0:
    print("うるう年です")
elif year % 100 == 0:
    print("うるう年ではありません")
elif year % 4 == 0:
    print("うるう年です")
else:
    print("うるう年ではありません")

# 3. 運転可能判定
age = 25
has_license = True
if age >= 18 and has_license:
    print("運転できます")
else:
    print("運転できません")
```
