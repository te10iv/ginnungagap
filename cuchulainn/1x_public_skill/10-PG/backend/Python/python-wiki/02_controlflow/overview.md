# 概要

## 1. 要点まとめ

- 制御構文はプログラムの実行の流れを制御する
- 条件分岐（if文）で条件に応じた処理を実行できる
- 繰り返し処理（for文、while文）で同じ処理を繰り返し実行できる
- `break`でループを中断、`continue`で次の繰り返しにスキップできる
- インデントがブロック構造を表すため、制御構文では特に重要

## 2. 詳細解説

制御構文は、プログラムの実行の流れを制御するための構文です。Pythonには主に3つの制御構文があります。

条件分岐は`if`文を使用します。条件が`True`のときと`False`のときで異なる処理を実行できます。`elif`で複数の条件を分岐させ、`else`でそれ以外の場合を処理できます。

繰り返し処理には`for`文と`while`文があります。`for`文はシーケンス（リスト、文字列など）の各要素に対して処理を繰り返します。`while`文は条件が`True`の間、処理を繰り返します。

ループの制御には`break`と`continue`があります。`break`はループを完全に終了させ、`continue`は現在の繰り返しをスキップして次の繰り返しに進みます。

Pythonでは、これらの制御構文のブロックはインデントで表現されます。同じインデントレベルのコードが同じブロックとして扱われます。

## 3. サンプルコード

```python
# 条件分岐の例
age = 20
if age >= 18:
    print("成人です")
else:
    print("未成年です")

# 繰り返し処理の例
# for文
for i in range(5):
    print(i)  # 0, 1, 2, 3, 4

# while文
count = 0
while count < 5:
    print(count)
    count += 1
```

## 4. 注意点・落とし穴

- インデントの統一が重要（スペース4つ推奨）
- 無限ループに注意（while文で条件が常にTrueにならないように）
- 条件式の比較演算子を正しく使う（`==`と`=`を混同しない）

## 5. 演習問題

1. 条件分岐を使って、数値が正か負か0かを判定するプログラムを作成してください
2. for文を使って、1から10までの数値を表示してください
3. while文を使って、カウントダウン（10から1まで）を表示してください

## 6. 解答例

```python
# 1. 正負の判定
num = 5
if num > 0:
    print("正の数です")
elif num < 0:
    print("負の数です")
else:
    print("ゼロです")

# 2. 1から10まで
for i in range(1, 11):
    print(i)

# 3. カウントダウン
count = 10
while count > 0:
    print(count)
    count -= 1
print("発射！")
```
