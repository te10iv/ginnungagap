# 関数の定義

## 1. 要点まとめ

- `def`キーワードで関数を定義する
- 関数名は変数名と同じ規則に従う
- 引数（パラメータ）を括弧内に指定できる
- 関数の処理はインデントされたブロックに記述する
- 関数は`return`文で値を返せる
- デフォルト引数を指定できる

## 2. 詳細解説

関数は`def`キーワードを使って定義します。`def 関数名(引数1, 引数2, ...):`の形式で記述し、その後のインデントされたブロックに処理を書きます。

関数名は変数名と同じ規則に従います。英数字とアンダースコア（`_`）を使用でき、数字で始めることはできません。また、予約語は使用できません。

引数（パラメータ）は、関数に渡す値です。引数は複数指定でき、関数内で変数として使用できます。引数にデフォルト値を指定することもでき、デフォルト値がある引数は呼び出し時に省略できます。

関数は`return`文を使って値を返すことができます。`return`がない場合、関数は`None`を返します。`return`が実行されると、関数の実行は終了し、その後の処理は実行されません。

関数を呼び出すには、関数名の後に括弧`()`を付け、必要に応じて引数を渡します。

## 3. サンプルコード

```python
# 基本的な関数定義
def say_hello():
    print("Hello!")

# 引数を受け取る関数
def greet(name):
    print(f"こんにちは、{name}さん")

# 複数の引数を受け取る関数
def add(a, b):
    return a + b

# デフォルト引数を持つ関数
def greet_with_default(name="ゲスト"):
    print(f"こんにちは、{name}さん")

greet_with_default()  # "こんにちは、ゲストさん"
greet_with_default("太郎")  # "こんにちは、太郎さん"

# 戻り値を返す関数
def multiply(x, y):
    result = x * y
    return result

# 複数の値を返す関数（タプルとして返す）
def get_name_and_age():
    name = "太郎"
    age = 25
    return name, age  # タプルとして返される

name, age = get_name_and_age()
print(f"{name}は{age}歳です")

# 早期リターン
def check_positive(num):
    if num <= 0:
        return False  # 早期リターン
    return True  # 正の数の場合のみ実行される
```

## 4. 注意点・落とし穴

- **関数の定義順序**: 関数は呼び出しより前に定義する必要があります。ただし、関数内で呼び出す場合は順序は関係ありません
- **引数の数**: 関数を呼び出す際は、定義された引数の数と一致させる必要があります（デフォルト引数がある場合は除く）
- **デフォルト引数の順序**: デフォルト引数は、デフォルト値のない引数の後に置く必要があります
- **return文の位置**: `return`文が実行されると、関数の実行は終了します。その後の処理は実行されません
- **変数のスコープ**: 関数内で定義した変数は、その関数内でのみ有効です（ローカル変数）

## 5. 演習問題

1. 引数を受け取らず、"Pythonを学んでいます"と表示する関数`study()`を作成してください
2. 2つの数値を受け取り、その積を返す関数`multiply(x, y)`を作成してください
3. 名前と年齢を受け取り、"私は[名前]です。[年齢]歳です"と表示する関数`introduce(name, age)`を作成してください
4. デフォルト引数を使って、挨拶文を表示する関数`greet(name="ゲスト")`を作成してください

## 6. 解答例

```python
# 1. study()関数
def study():
    print("Pythonを学んでいます")

study()

# 2. multiply()関数
def multiply(x, y):
    return x * y

result = multiply(5, 7)
print(result)  # 35

# 3. introduce()関数
def introduce(name, age):
    print(f"私は{name}です。{age}歳です")

introduce("山田太郎", 25)

# 4. デフォルト引数を持つgreet()関数
def greet(name="ゲスト"):
    print(f"こんにちは、{name}さん")

greet()  # "こんにちは、ゲストさん"
greet("太郎")  # "こんにちは、太郎さん"
```
