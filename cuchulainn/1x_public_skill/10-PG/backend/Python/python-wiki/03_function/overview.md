# 概要

## 1. 要点まとめ

- 関数は処理をまとめて再利用可能にする
- `def`キーワードで関数を定義する
- 関数は引数（パラメータ）を受け取れる
- 関数は戻り値（return値）を返せる
- 関数内で定義した変数はローカル変数（スコープが関数内）
- 関数を使うことでコードの可読性と再利用性が向上する

## 2. 詳細解説

関数は、処理をまとめて名前を付けたものです。同じ処理を何度も書く必要がなくなり、コードの再利用性と可読性が向上します。

Pythonでは`def`キーワードを使って関数を定義します。関数名の後に括弧`()`を書き、その中に引数（パラメータ）を指定できます。引数は関数に渡す値で、関数内で変数として使用できます。

関数は`return`文を使って値を返すことができます。`return`がない場合、関数は`None`を返します。`return`が実行されると、関数の実行は終了します。

関数内で定義した変数は、その関数内でのみ有効なローカル変数です。関数の外で定義した変数はグローバル変数で、関数内から参照できますが、変更するには`global`キーワードが必要です。

関数を使うことで、コードを論理的な単位に分割でき、デバッグやテストが容易になります。

## 3. サンプルコード

```python
# 基本的な関数定義
def greet():
    print("こんにちは")

# 関数の呼び出し
greet()

# 引数を受け取る関数
def greet_name(name):
    print(f"こんにちは、{name}さん")

greet_name("太郎")

# 戻り値を返す関数
def add(a, b):
    return a + b

result = add(3, 5)
print(result)  # 8
```

## 4. 注意点・落とし穴

- 関数の定義は呼び出しより前に書く必要がある
- 引数の数と型を正しく渡す
- 戻り値がない場合は`None`が返る

## 5. 演習問題

1. 引数を受け取らず、"Hello, World!"を表示する関数を作成してください
2. 2つの数値を受け取り、その和を返す関数を作成してください
3. 名前を受け取り、"こんにちは、[名前]さん"と表示する関数を作成してください

## 6. 解答例

```python
# 1. Hello, World!を表示
def hello():
    print("Hello, World!")

hello()

# 2. 2つの数値の和
def add(a, b):
    return a + b

result = add(10, 20)
print(result)  # 30

# 3. 名前を表示
def greet(name):
    print(f"こんにちは、{name}さん")

greet("太郎")
```
